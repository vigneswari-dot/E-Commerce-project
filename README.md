# WEEK 1 
# E-Commerce Order Management Database System

## Project Overview

This repository contains the Week 1 deliverables for the **E-Commerce Order Management Database System** semester project. The project focuses on analyzing business requirements and preparing the necessary documentation before designing the database.

The proposed system manages customer registration, product management, inventory tracking, order processing, payment management, shipment tracking, and customer reviews using a centralized database.

## Objectives

* Analyze the business requirements of the proposed system.
* Identify stakeholders and their roles.
* Define functional and non-functional requirements.
* Prepare a Software Requirement Specification (SRS).
* Establish the foundation for future database design.
